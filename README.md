# 🎯 AI Interview Preparation System

AI Interview Preparation System is a full-stack AI-powered interview practice platform that helps users prepare for technical interviews through resume-based question generation, AI evaluation, and performance analysis.

## 📖 Overview

Features include:

- 🤖 AI-powered interview question generation using Google Gemini
- 📄 Resume Upload & Analysis (PDF/DOCX)
- 💬 Interactive Mock Interview Sessions
- 📊 AI-based Performance Evaluation
- 📈 Interview Reports & Analytics
- 📝 Interview History Management
- 💾 SQLite Database Integration
- 📱 Clean & Responsive Streamlit Interface
